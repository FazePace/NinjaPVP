# NinjaTiers - Official Web Leaderboard

Official web interface for the NinjaPVP Minecraft Server. This repository contains the frontend and the Netlify Functions backend for real-time synchronization.

## Features
- **Real-time Ranking**: Automatically synchronized with the Minecraft server.
- **Glassmorphism Design**: High-end premium aesthetic.
- **Auto-Update**: Live-polling implementation for seamless data refreshing.

## Deployment
Hosted on Netlify with CI/CD integration from this repository.
